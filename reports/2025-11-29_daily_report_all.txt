[6] 내일부터 실행할 구체적 행동 가이드 5가지

1. **손실 제한 규칙 강화**  
   - 단일 트레이드 손실이 -2%를 초과하면 자동 청산 및 해당 심볼 당일 추가매매 금지(rule: if trade_loss_pct <= -2, force_exit & disable_symbol_today).
2. **진입 신호 확률 필터 적용**  
   - ML 예측 확률 0.5 미만에서는 신규 진입 금지(rule: if ml_proba < 0.5, entry_allowed = 0).
3. **변동성 급등 직후 진입 제한**  
   - pre_window의 volatility_pct가 1% 이상인 경우 신규 진입 제한(rule: if pre_window.volatility_pct >= 1, entry_allowed = 0).
4. **TP1 도달 시 50% 부분청산 자동화**  
   - 목표가(1%) 도달 시 진입 수량의 50% 자동 부분청산(rule: if profit >= 1%, auto_partial_exit = 50%).
5. **진입 후 3분 이내 손실 -0.5% 이상 시 즉시 손절**  
   - 진입 직후 불리한 흐름이면 신속히 손실 제한(rule: if holding_time < 3min & profit_pct <= -0.5%, force_exit).
